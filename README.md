# Image Processing with C++ and OpenCV

This repository contains image processing exercises, laboratory work, and camera interfacing experiments developed as part of a **Master’s course** in **Information and Communication Technology** at **HAW Hamburg**.

The project focuses on **image acquisition, processing, and analysis using C++ and OpenCV**, with practical experiments.

##  Folder Description

### `Camera_AlliedVisionAlvium/`

This directory focuses on **working with Allied Vision Alvium 1800 USB cameras** using **C++ and OpenCV**.

The main goals are:

* Capturing image frames using OpenCV
* Configuring and experimenting with camera parameters
* Understanding industrial camera behavior in real-world conditions

---

### `Exercise/`

* Contains **programming exercises**
* Focuses on:

  * OpenCV fundamentals
  * Classic Image Processing Steps

* Exercises are aligned with the course workbook and lectures

---

### `ImagingData/`

* Contains **image datasets provided by the professor**
* Used for:

  * Lab tasks
  * Workbook exercises
  * Offline image processing experiments

---

### `Labs/`

* Contains implementations from **4 laboratory sessions**
* Each lab includes **three main tasks**
* Tasks involve:

  * Image filtering
  * Feature extraction
  * Camera-based experiments
* Detailed task descriptions are available in `Lab_Tasks.pdf`

---

### `Lab_Tasks.pdf`

* Official **lab task descriptions**
* Provided by the course instructor
* Defines objectives, requirements, and expected results for each lab session
